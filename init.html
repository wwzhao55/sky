#安装

将代码部署到域名（例scrm.dataguiding.com/game/）指向的目录

使用的域名（例scrm.dataguiding.com）需要在公众号中设置（登录公众号后台—右上角公众号名称—功能设置—JS接口安全域名）

替换文件index.html，game.min.js中的域名为要使用的域名

##初始化数据库

```
mysql -uroot -p < init.sql
```
之后输入数据库密码即可

##修改全局信息

对于game目录下的global_info.php，需要配置如下字段



$appid = '公众号的appid'; 

$appsecret = '公众号的秘钥';

/* 重定向的地址，应为最终能方位到game目录下oauth.php的路径 */

$redirect_url = "./game/oauth.php";

/*mysql数据库的host*/

$host = "localhost";

$username = "mysql数据库的用户名";

$password = "mysql数据库的密码";

$dbname = "balloon_game";//不修改

$random_break_point = 1000;//生成随机爆点的最高值
